<completed_work>

- **Task 1 (done):** Three-part fix to `renderHBadges`:
  1. **Explicit-H awareness**: `implicitCount = totalCount − bonded-explicit-H-count`. Skip atom if `implicitCount ≤ 0` (all H already drawn). Previously skipped on ANY explicit H, now only skips when ALL H are explicit.
  2. **Per-atom residual count**: Badge text/color derived from `implicitCount` per atom, not from the layer total. A methyl with 2 explicit H + 1 implicit shows "H" (not "H3").
  3. **Unified angular-gap placement**: For each heavy atom, collect `atan2` angles to ALL bonded neighbors visible in the SVG (heavy atoms via bonds + explicit H atoms). Sort, find the largest angular gap, place badge at gap midpoint. This avoids both bond lines and drawn H labels in one pass, replacing the earlier avoidDir + centroid-fallback approach.
  - Fallback: centroid-based radial offset when no neighbor SVG positions are available (struct absent or neighbors not in DOM).
  - All 171 tests passing.
</completed_work>
