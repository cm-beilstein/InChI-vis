## Interaction Contract

### Sub-token hover states (h-layer)

Phase 8's primary new interaction is per-comma-group hover within the h-layer. The sub-token rendering is **already implemented** in `LayerText.tsx ‚Üí HLayerText`. The interaction contract is:

| User action | Visual result |
|-------------|---------------|
| Hover h-layer chip (no sub-token) | Whole layer activates; all H-bearing heavy atoms highlighted by count color; layer chip background `--c-hydro-bg` |
| Hover fixed-H sub-token (e.g. `7H2`) | Targeted atoms highlighted in `--c-hydro-{count}` color; H-count SVG badge appears above each atom (`H2` etc.); sub-token ring `inset 0 0 0 1.5px currentColor` |
| Hover mobile-H sub-token (e.g. `(H,3,4)`) | All candidate atoms highlighted in `--c-hydro-mobile`; italic `H?` SVG badge above each; italic sub-token text |
| Hover explicit-H sub-token (maps to `hAtomPoolIds`) | H atom highlighted + bonded heavy atom highlighted + bond between them highlighted; all in `--c-hydro-{count}` |
| Mouse-out from any sub-token | Sub-hover cleared; reverts to layer-wide highlight if h-layer still active |
| Mouse-out from h-layer chip | All highlights cleared; canvas returns to default |

### SVG badge lifecycle

| Event | Action |
|-------|--------|
| `hAtoms` or `mobileH` subHover set | `highlights.create()` ‚Üí then inject SVG `<text>` elements with `data-h-badge="true"` |
| Any hover-out (subHover cleared) | Remove all `[data-h-badge]` elements from SVG root; same path as `whiteAtomLabels` cleanup |
| `highlights.clear()` | Ketcher recreates SVG elements, clearing any injected badges automatically (belt-and-suspenders: also remove `[data-h-badge]` explicitly) |

### Badge visual specification

- Element: SVG `<text>` injected into Ketcher's `render.paper.canvas` element
- Attribute: `data-h-badge="true"` (for targeted cleanup)
- Position: atom center `x`, atom center `y + 20` (below atom label, matching design handoff `canvas.jsx:203`)
- Font size: 12px (matching `.molecule .atom-num` register)
- Font weight: 500
- Text anchor: `middle`
- Dominant baseline: `central`
- Fill: resolved CSS variable value of `--c-hydro-{count}` (for fixed-H) or `--c-hydro-mobile` (for mobile-H)
- Pointer events: none
- Content: `H` (count=1), `H2` (count=2), `H3` (count=3), etc. ‚Äî plain ASCII digits, no Unicode subscripts (SVG text, not HTML)
- Mobile-H badge content: `H?` in italic (`font-style: italic`)

Source: design handoff `canvas.jsx` lines 196‚Äì246; existing `whiteAtomLabels` pattern in `useKetcherHighlights.ts`.
