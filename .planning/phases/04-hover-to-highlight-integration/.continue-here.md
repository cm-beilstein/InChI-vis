# BLOCKING CONSTRAINTS — Read Before Anything Else

> These are not suggestions. Each constraint below was discovered through failure.
> Acknowledge each one explicitly before proceeding.

- [ ] CONSTRAINT: oklch-color-incompatibility — Ketcher's Raphaël SVG renderer cannot parse `oklch()` color strings returned by `getComputedStyle`. `resolveVar()` in `src/lib/highlightUtils.ts` was fixed to normalise any CSS color to `rgb()` via a 1×1 canvas. This fix is **uncommitted** and must be committed before running UAT — otherwise highlights will render with wrong/no color.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| oklch → Raphaël incompatibility | `getComputedStyle` returns `oklch()` strings for CSS custom properties. Raphaël (Ketcher's SVG renderer) silently rejects these — highlights created with oklch colors appear transparent or black. | blocking | `resolveVar()` now normalises via canvas; never pass raw `getComputedStyle` output to Ketcher highlight API |
