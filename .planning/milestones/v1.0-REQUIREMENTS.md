## Core Value

Every chunk of an InChI string is hoverable, explained, and linked back to the atoms in the drawing — demystifying a notation that most chemists treat as opaque.

## v1 Requirements — All Shipped

### Editor

- [x] **EDIT-01**: User can draw and edit molecules using the embedded Ketcher standalone editor (no backend required) — Phase 1 ✓
- [x] **EDIT-02**: User can click a preset molecule row in the sidebar to load it into the canvas — Phase 5 ✓
- [x] **EDIT-03**: Canvas overlay shows the current molecule name, molecular formula, and heavy-atom count — Phase 5 ✓

### InChI Display

- [x] **INCHI-01**: InChI string updates live on every structure change (debounced ≤150ms) using Ketcher's built-in WASM InChI library — Phase 2 ✓
- [x] **INCHI-02**: Each InChI layer chunk is rendered in its design-token accent color, separated by `/`, with sub-spans for formula elements, H-counts, and stereo parities — Phase 3 ✓
- [x] **INCHI-03**: Hovering a layer chunk highlights the matching atoms and bonds in the Ketcher canvas and dims all other layer chunks — Phase 4 ✓
- [x] **INCHI-04**: Hovering a sub-token (element symbol, atom number, parity token, H-group) overrides the layer-wide highlight with a more targeted one, suppressing the layer-wide fallthrough entirely — Phase 4 ✓
- [x] **INCHI-05**: Hovering H sub-token highlights explicit hydrogen atoms in Ketcher canvas — Phase 6 ✓ *(added during development)*
- [x] **INCHI-06**: Multi-fragment molecules highlight correct atoms/bonds per fragment on layer hover — Phase 7 ✓ *(added during development)*
- [x] **INCHI-07**: Hovering p-layer highlights protonation-site atoms (q-layer or heteroatom fallback) — Phase 7 ✓ *(added during development)*
- [x] **INCHI-08**: Per-group h-layer sub-tokens with implicit H badge, explicit H bond highlight, and mobile-H group highlight — Phase 8 ✓ *(added during development)*

### Explanation

- [x] **EXPL-01**: Left explanation card shows the currently hovered layer's prose description and a molecule-specific reading-code block — Phase 3 ✓
- [x] **EXPL-02**: Right legend card lists all 11 InChI layer types with color swatches; hovering a row reveals a tooltip with fuller description and example syntax — Phase 3 ✓
- [x] **EXPL-03**: Idle state (no layer hovered) shows the default explanation card content — Phase 3 ✓

### Mapping

- [x] **MAP-01**: Mapping strip below the InChI display shows each atom's Ketcher index → canonical InChI index; identity pairs are dimmed, divergent pairs are highlighted green — Phase 5 ✓
- [x] **MAP-02**: Footnote strip shows the InChI definition and keyboard hint legend (Hover, Click) — Phase 5 ✓
- [ ] **MAP-03**: Current molecule state encoded in URL (hash or query param) — **DEFERRED to v2** (decided in Phase 6 planning; STATE.md TODOs)

### Polish

- [x] **PLSH-01**: Empty canvas or invalid/disconnected structure shows a placeholder message in the InChI display instead of an error — Phase 6 ✓
- [x] **PLSH-02**: WASM initialisation loading state is shown in the editor panel until Ketcher is ready — Phase 1 ✓
- [x] **PLSH-03**: Typography (IBM Plex Sans/Serif/Mono), color tokens, spacing, and hover transitions match the design handoff pixel-for-pixel — Phase 3 ✓
- [x] **PLSH-04**: Copy-to-clipboard button copies verbatim InChI string with visual confirmation — Phase 7 ✓ *(added during development)*

## Out of Scope (Confirmed at v1.0)

| Feature | Reason |
|---------|--------|
| Backend / Indigo server | ketcher-standalone runs entirely in-browser via WASM |
| User accounts / persistence | Stateless educational tool; URL sharing covers the core use case |
| 3D structure viewer | InChI is a 2D notation; 3D adds scope without illuminating InChI |
| Database / substructure search | Shifts product identity away from the explainer tool |
| Mobile-native app | Web-first; Ketcher canvas is not well-suited to touch input |
| Language localisation | English-only for v1; deferred |
